# Bank Customer Churn Analysis

This project analyzes customer churn behavior using SQL and Power BI to identify key factors influencing customer attrition. The analysis helps uncover customer segments at risk of leaving and provides actionable insights to improve retention strategies.

## Key Features

- Analyzed 10,000 customer records using SQL.
- Calculated churn rate and customer retention metrics.
- Identified churn patterns by age, gender, and geography.
- Evaluated the impact of products, balance, and activity status.
- Built an interactive Power BI dashboard with dynamic filters.
- Generated business recommendations to reduce churn.

## Key Insights

- Overall churn rate is 20.37%.
- Germany shows the highest churn rate.
- Female customers churn more than males.
- Inactive customers are more likely to leave.
- Customers with 3–4 products exhibit higher churn.
- Senior customers show elevated churn risk.

## Tools Used

- SQL (PostgreSQL)
- Power BI
- Excel / CSV
